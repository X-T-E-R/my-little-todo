# Brand Assets

![Primary logo](./logo-primary.svg)
![Generated logo mark](./logo-mark-generated.png)

这一版品牌资产围绕你选中的 `MLT` 方标展开，没有再引入额外符号。重点是把项目原本那种“不是记账，而是把人往前推”的气质收得更明确一点。

## Primary Slogan

- English: `Move, don't just list.`
- 简体中文：`别只列清单，开始推进。`

## Supporting Line

- English: `Your external execution system.`
- 简体中文：`你的外部执行系统。`

## Rationale

- 图标主体保留 `MLT` 的读感，适合桌面托盘、PWA 和移动端启动图标。
- 右下角的绿色圆点不是装饰，它代表 “now” 的触发点，强调先动起来。
- 整体色彩继续沿用项目原本的 dusty blue，但图形比例回到你选中的那一版，尤其保留偏小的 `T` 和右下角的 now dot。

## Files

- `logo-mark.svg`：图标版，适合 favicon、app icon、tray icon。
- `logo-mark-generated.png`：按参考图重做的透明底正方形 PNG，适合需要位图对齐的场景。
- `logo-primary.svg`：横向正式 logo，适合文档首页、发布页、宣传素材。
- `banner-en.png`：英文文档横幅，白底紧凑版。
- `banner-zh-CN.png`：中文文档横幅，白底紧凑版。
